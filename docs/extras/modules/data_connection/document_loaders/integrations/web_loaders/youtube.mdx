# YouTube transcripts

This covers how to load youtube transcript into LangChain documents.

## Setup

You'll need to install youtube-transcript library:

```bash npm2yarn
npm install youtube-transcript
```

## Usage

You need to specify a link or video id in the `urlOrId`. You can also specify `language` in [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes).
